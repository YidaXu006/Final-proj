# 波的叠加与干涉交互式演示工具 使用说明
## 一、工具简介
本工具为控制台交互式程序，无需复杂环境，输入参数即可实时绘制两列简谐波的叠加干涉波形，直观展示相长、相消干涉等物理现象，内置参数校验、快捷指令等功能，操作简单。

## 二、运行准备
1. 环境要求：Python 3.6及以上版本，Windows/macOS/Linux系统均可。
2. 安装依赖：打开终端，输入指令安装numpy和matplotlib库。
3. 启动工具：将工具代码保存为.py文件，终端进入文件目录，执行启动指令运行程序。

## 三、核心操作
### 1. 指令选择
启动后首先显示指令提示，可输入以下指令：
- q：退出程序，关闭所有波形图；
- reset：重置所有参数为初始默认值（A1=1.0、A2=1.0、相位差=0.0、波长=4.0）；
- help：查看快捷参数模板，快速获取相长/相消干涉的典型参数；
- 直接回车：跳过指令环节，进入参数输入界面。

### 2. 参数输入
按顺序输入4个物理参数，每个参数可直接输入数值，或回车使用当前默认值：
- 波1振幅（A1）：取值0.1~3.0，控制第一列波的峰值高度；
- 波2振幅（A2）：取值0.1~3.0，控制第二列波的峰值高度；
- 相位差（Δφ）：取值0~6.28（2π），核心干涉参数，输入0为完全相长干涉，输入3.14（π）为完全相消干涉；
- 波长（λ）：取值2.0~8.0，控制波形疏密程度，值越小波形越密。

### 3. 输入校验
- 有效输入：控制台提示“输入有效/使用默认值”，自动进入下一步；
- 无效输入：输入非数字或数值超出范围时，控制台提示错误，循环等待重新输入，直至正确。

### 4. 波形查看
参数输入完成后，控制台提示绘制中，随后弹出波形图窗口：
- 蓝色实线为波1，红色虚线为波2，绿色粗线为叠加波；
- 图表标题显示当前参数，网格和零基准线辅助观察；
- 无需关闭波形图，可直接返回控制台输入新参数，对比不同波形效果。

### 5. 参数缓存
每次输入的参数会自动作为下一次的默认值，输入reset可恢复初始默认值。

## 四、典型操作示例
1. 完全相长干涉：波1、波2振幅均用默认值1.0，相位差输入0，波长用默认值4.0，叠加波振幅最大；
2. 完全相消干涉：波1、波2振幅均用默认值1.0，相位差输入3.14，波长用默认值4.0，叠加波振幅为0；
3. 快捷参数使用：输入help查看模板参数，按模板数值输入，快速验证不同干涉效果。

## 五、注意事项
1. 若波形图无中文显示，需确保系统安装中文字体；
2. 退出程序前可手动关闭波形图，或输入q自动关闭；
3. 参数仅需输入数字，无需输入单位；
4. 若绘制卡顿，可调大波长值减少波形密集度。

## 六、常见问题解决
1. 启动时报模块缺失：重新安装numpy和matplotlib库；
2. 无波形图弹出：调整matplotlib后端配置；
3. 提示变量相关错误：使用修正版代码，确保全局变量声明正确。
