# 物理实用编程期末作业

## 项目概述
本项目是一个综合性的物理实用编程项目，涵盖了波动力学、量子力学、数据分析等多个物理领域的编程实现。项目采用模块化设计，将不同的物理主题和计算方法分成多个独立部分。

## 项目结构与内容

### part_1：
#### M1.md：三维矢量场与静电场计算
- 三维矢量场叉积、旋度计算及恒等式验证的Mathematica编程分析
- 确认Cross函数计算叉积时严格遵循三维右手定则的分量展开
- 验证Curl函数默认采用笛卡尔坐标系及其行列式展开逻辑
- 验证∇×(∇φ)=0恒等式的符号运算过程
- 三个点电荷系统静电场计算的Mathematica编程实现
- 解决变量命名冲突、符号运算简化和电场零点求解等问题

#### M2.md：阻尼谐振子与受迫振动分析
- 一维阻尼谐振子的解析解与可视化的Mathematica编程
- DSolve函数的参数格式要求和导数的标准写法
- 变量作用域管理和空解容错处理
- 可视化图形对象生成的底层逻辑和避坑方法
- 简单受迫振动的共振特性分析和瞬态-稳态过渡过程模拟
- 解析公式封装化、数值求解物理化和符号运算规范化的编程实践

### part_2：
#### Q1.md：傅里叶展开Python实现
- 傅里叶级数的物理意义和方波、三角波的展开分析
- quad积分精度控制和x周期化处理的必要性
- 吉布斯现象的观察方法和过冲特性分析
- 方波和三角波收敛速度的量化比较（MSE误差分析）
- FFT幅度谱计算逻辑和stem函数的版本兼容性
- 函数缓存机制和冗余计算在教学科研中的价值

#### Q2.md：Python数据处理技巧
- 批量读取CSV文件的编码处理方法
- 缺失值处理中的均值填充和中位数填充适用场景
- 数据标准化和归一化的核心区别及选择依据
- 大文件分块处理的内存占用和效率平衡策略
- 重复数据清洗的精细处理方式
- 时间序列数据插值填补缺失值的注意事项

### part_3：
#### Q3.md：抛体运动Python实现
- 无阻力和有阻力抛体运动的轨迹计算、能量分析和动画实现
- 有无阻力抛体运动的物理差异（运动形式、机械能变化、轨迹形状、速度变化）
- 有阻力时最佳发射角偏离45°的物理原因分析
- set_data方法的参数要求和索引越界问题解决
- 机械能守恒验证的容差设置和有阻力能量分析的完善
- FuncAnimation的blit参数工作机制和Tkinter渲染异常规避

#### Q4.md：数据可视化Python实现
- 基础图表绘制和数据筛选功能的实现
- 静态可视化和动态可视化的设计原则差异
- 动态可视化更新频率过高降低可读性的原因分析
- set_text错误与信号连接异常的解决方法
- 坐标转换误差的阈值设置和复杂动态图表的信息补充
- 动态可视化与Matplotlib交互工具的协同机制优化

### part_4：
#### 核心功能
- 控制台交互式程序，可实时绘制两列简谐波的叠加干涉波形
- 支持自定义波1振幅、波2振幅、相位差和波长等参数
- 直观展示相长干涉、相消干涉等物理现象
- 内置参数校验、快捷指令和帮助功能

#### 使用说明
- 启动程序后可输入q退出、reset重置参数、help查看快捷模板
- 按顺序输入波1振幅、波2振幅、相位差和波长等参数
- 程序会实时绘制波形图，展示两列波的叠加效果

### part_5：
#### quantum_wavefunc_redis：量子波函数Redis存储系统
- 基于Redis数据库的量子波函数存储系统
- 支持波函数的增删改查等基本操作
- 利用Redis的高并发、低延迟特性，适合处理量子力学中的实时计算需求
- 包含完整的项目结构（pyproject.toml、README.md、核心代码目录）

## 技术栈
- Python 3.6+
- NumPy：数值计算
- Matplotlib：数据可视化
- SciPy：科学计算
- Redis：数据存储（part_5）
- Jupyter Notebook：交互式计算（.nb文件）
- Mathematica：符号计算（part_1）
